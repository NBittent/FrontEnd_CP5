# Frontend Application Documentation

This repository contains the source code for the frontend application developed as part of CP5.

## Project Overview

The frontend application is designed to provide an intuitive user interface for users to interact with our services. It is built using modern web technologies.

## Features
- Responsive design
- User authentication
- REST API integration
- Interactive UI components

## Getting Started

To get a local copy up and running, follow these simple steps:

1. **Clone the repo**
   ```sh
   git clone https://github.com/NBittent/FrontEnd_CP5.git
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Run the application**
   ```sh
   npm start
   ```

## Technologies Used
- HTML
- CSS
- JavaScript
- React

## Deployment

This project is configured to be deployed via GitHub Pages from the `main` branch with `index.html` as the entry point.
