# Documentação do Projeto FrontEnd_CP5

## Descrição
Este é um projeto de FrontEnd desenvolvido como parte do CP5.

## Instruções de Uso
Siga os passos abaixo para configurar o ambiente:

1. Clone o repositório: `git clone https://github.com/NBittent/FrontEnd_CP5`
2. Navegue até o diretório do projeto.
3. Instale as dependências: `npm install`
4. Para executar o projeto, use: `npm start`
